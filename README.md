# Neurolume - Herbal Capsule for Stress

A premium, interactive, and fully responsive landing page for **Neurolume (Herbal Capsule for Stress)**, featuring a modern botanical wellness aesthetic. Built using React 19, Vite 8, Tailwind CSS v4, and Framer Motion.

## Key Features
- **Interactive Stress Response Simulator**: Allows users to adjust lifestyle sliders (stress, sleep, fog) and toggle Neurolume to see its biochemical impact on Adrenal Cortisol, GABA, and Serotonin.
- **Active Ingredient Codex**: Clickable cards for each of the 5 bio-active extracts (Shankhpushpi, Ashwagandha, Tagar, Jayphal, Basant) detailing dosage ratios and specific neuroprotective mechanisms.
- **Prescribing Information Monograph**: Tabbed accordion interface detailing indications, dosage timing, safety/overdose protocols, and packaging.
- **Aesthetic Design**: Forest/moss green base with glowing sunset amber accents, micro-particles, leaf masks, and smooth page load-in animations.

## Tech Stack
- **Framework**: React 19 & Vite 8
- **Styling**: Tailwind CSS v4 & custom glassmorphism layers
- **Animations**: Framer Motion

## Installation & Running

1. **Install Dependencies**:
   ```bash
   npm install
   ```
2. **Start Dev Server**:
   ```bash
   npm run dev
   ```
3. **Build for Production**:
   ```bash
   npm run build
   ```
